Credits

Third-party data services, datasets, and software used by Pi-Sky. Thank you to the authors and providers.

  • adsb.lol ADS-B API: flight routes (routeset), live aircraft positions, and aircraft metadata (type, description, registration) by ICAO hex. https://api.adsb.lol/docs
  • OpenFlights Airport and airline reference data (cached locally as airports.dat / airlines.dat) for coordinates, countries, and airline names. https://openflights.org/data.html
  • Mictronics Aircraft Database Optional local aircraft registration / ICAO type / model strings from user-uploaded dump1090-style exports. https://www.mictronics.de/aircraft-database/#
  • OpenStreetMap & Nominatim Map tiles and reverse geocoding (place names from coordinates). OpenStreetMap data is © OpenStreetMap contributors. https://www.openstreetmap.org/copyright
  • Wikimedia Commons API Search API used to find aircraft photos by registration where applicable. https://commons.wikimedia.org/wiki/Commons:API
  • Google Favicons Favicon lookup for airline logo icons derived from carrier domains (where available). Google favicon service
  • PlaneSpotters.net (replay UI) Optional aircraft photo lookup in the replay map view. https://www.planespotters.net
  • Leaflet & Anime.js Interactive maps (Leaflet) and UI animation (Anime.js, via CDN) on map and list views. Leaflet · Anime.js
  • FlipOff Split-flap display UI on the flight list (card view). https://github.com/magnum6actual/flipoff
